# 基金投资实用指南

## 基金类型

| 类型 | 风险 | 常见场景 |
|---|---|---|
| 货币基金 | 低 | 现金管理，如余额宝/零钱通 |
| 纯债基金 | 中低 | 稳健底仓，波动通常小于股票类基金 |
| 混合债基 | 中 | 稳中求进，但含权益或转债时波动会上升 |
| 宽基指数基金 | 中 | 长期定投核心，如沪深300、中证500、创业板、纳指等 |
| 行业主题基金 | 中高 | 医药、白酒、半导体、新能源等，波动大，比例不宜过高 |
| 主动混合/股票基金 | 中高/高 | 依赖基金经理能力，需关注风格稳定性和回撤 |
| QDII/海外基金 | 中高 | 全球分散，但有汇率、额度、估值时差等影响 |

## 定投原则

1. 用 3 年以上不用的闲钱。
2. 普通人优先考虑宽基指数或稳健均衡型产品。
3. 行业主题基金不要重仓押注，避免单一赛道风险。
4. 定投不是稳赚，核心是用时间和分批买入降低择时压力。
5. 短期亏损时先看是否符合原计划，不要只因情绪追涨杀跌。

## 资产配置思路

- 保守型：现金/货币 + 债券为主，少量指数和黄金。
- 稳健型：债券与宽基指数做核心，少量行业和海外分散。
- 激进型：权益类比例更高，但必须接受更大回撤。

## 再平衡

再平衡是把涨太多的部分适当减一点，把跌得多但仍符合长期逻辑的部分补一点，让组合回到目标比例。

- 检查频率：每季度或每半年。
- 触发条件：某类资产偏离目标比例 5% 以上。
- 优先方式：用新增资金补低配资产，少做频繁卖出。

## 避坑

- 不要借钱买基金。
- 不要只看近 1 个月收益排行。
- 不要把所有钱押在单一行业主题。
- 不要忽略费率和持有期限。
- 不要把短期要用的钱投进高波动基金。
- 任何模型测算都只是情景假设，不代表未来收益。
