# Git Notes

## Repository Purpose

Personal CowWeCom deployment and development workspace for WeCom Bot integration, OpenAI-compatible model providers, image generation tooling, scheduler tools, and agent memory features.

## Ignore And Secret Policy

Runtime secrets and local state stay out of Git. The repository ignores `config.json`, env files, key/certificate files, Weixin/WeCom credentials, cookies, tokens, session files, logs, virtual environments, generated build output, generated knowledge indexes/originals/reports, `.codex/`, `.playwright-mcp/`, and other local workspace artifacts.

Do not commit API keys, QR login material, credential JSON files, chat logs, cookies, tokens, or local auth material. Use committed templates and documentation only for placeholders and safe defaults.
